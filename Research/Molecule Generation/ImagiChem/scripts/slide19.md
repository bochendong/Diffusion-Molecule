Stage 1 establishes the foundation: seed derivation and image preprocessing. Here, the image is normalized, and a seed is produced so that the same visual input leads to the same downstream behavior. 

Stage 2 turns visual patterns into basic chemical alphabet. Instead of just using pixels as data, the method using features like brightness and texture, and maps them into chemical components.

Stage 3 builds the molecule step by step. First, it creates a basic structure, and then adds functional groups in a controlled way. The key point is that this is not random—it follows a structured process guided by the image

Stage 4 is the quality check. It tests whether the molecules are valid, ranks the better ones, and prepares them for further use. So even if earlier steps generate candidates, this stage makes sure the final results are reliable.
